스탬프 콘텐츠
입력 가이드

엑셀 입력폼에 스탬프존(장소·소개·좌표) 정보를 적고, 이미지와 함께 묶어 한 번에 등록하는 방법을 처음부터 끝까지 안내합니다. 스탬프투어는 위치기반(GPS) 인증이라 위도·경도·반경이 필수입니다.

📥 샘플폼 · 인천 📥 예시폼 · 전국 시장(코스 포함) 작성 방법 보기

※ 두 가지 샘플을 제공합니다 — 인천(기본 · 코스 없음) / 전국 시장(코스를 시장별로 묶은 예시).

✓

전체 과정은 네 단계입니다. 입력폼을 작성하고, 이미지를 준비해, 하나의 ZIP으로 묶어 일괄 등록합니다.

2

3
ZIP 묶기
엑셀 + images

일괄 업로드
관리자에서 ZIP 등록
💡 참고 이미지가 없어도 등록 가능합니다. 사진이 없는 스탬프존은 ‘이미지 준비중’ 기본 이미지가 자동으로 들어갑니다. 등록은 관리자 [스탬프 › 스탬프 일괄등록]에서 진행합니다.
1

입력폼(엑셀) 작성

샘플 입력폼을 내려받아, 위의 형식 그대로 고쳐 쓰면 됩니다. 1행(항목 이름)은 그대로 두고 2행(예시)부터 스탬프존을 한 줄씩 입력합니다. 입력한 줄 순서가 곧 앱 노출 순서입니다.

항목설명
번호필수 1부터 차례대로 매기는 번호. 화면 노출 순서이자 이미지 파일명과 연결됩니다. 예) 1, 2, 3, 4 …
장소필수 앱에 표시되는 장소 이름. 다국어가 필요하면 한글명(영문명) 형식으로 적습니다. 예) 인천시립박물관 · 센트럴파크(centralpark)
한줄소개선택 목록·상단에 보이는 짧은 소개 또는 분류 라벨. 예) 박물관 · 문화재 · 공원
장소 소개필수 상세 페이지 본문 설명. 외부 링크(URL)를 넣으면 클릭 시 해당 페이지(웹·유튜브·인스타 등)로 이동합니다. 예) 1946년 개관한 우리나라 최초의 공립박물관… / https://example.com
스탬프존 — 위치기반(GPS) 인증 기준 · 세 항목 모두 필수
위도필수 지도 표시 + GPS 인증 기준 좌표(Latitude). 소수점 6자리 이상 권장. 예) 37.420809
경도필수 지도 표시 + GPS 인증 기준 좌표(Longitude). 소수점 6자리 이상 권장. 예) 126.653244
반경필수 좌표를 중심으로 한 스탬프 인증 반경(미터). 이 원 안에 들어오면 스탬프를 획득합니다. 넓은 장소는 크게, 작은 지점은 작게 설정합니다. 예) 공원·해변 100 / 일반 건물 50 / 작은 정자·묘역 20
코스선택 스탬프존을 여러 묶음(코스)으로 나누는 경우에만 입력합니다. 자세한 사용법은 아래 코스 기능을 참고하세요. 예) 1코스 · 남대문시장(namdaemunmarket)

⚠️ 작성 시 유의할 점
  • 장소 · 장소 소개 · 위도 · 경도 · 반경은 반드시 채워야 합니다(비면 등록되지 않음).
  • 번호는 1부터 빠짐없이 이어지게 하고, 이미지 파일명(1.png, 1_2.png …)과 일치시킵니다.
  • 위도·경도는 소수점 6자리 이상, 반경은 미터(숫자)로 입력합니다.
  • 장소명이 빈 줄은 자동으로 건너뜁니다. 스탬프존 추가는 줄(행)을 추가해 입력합니다.
  • 1행(항목 이름)은 지우거나 바꾸지 마세요.
🔗 관리자 입력 항목과 1:1 일치 — 번호→노출 순서, 장소→장소명(다국어), 한줄소개→부제, 장소 소개→소개, 위도·경도·반경→스탬프존, 코스→코스. 관리자 [스탬프 › 스탬프 일괄등록]이 헤더명을 자동 인식해 그대로 등록합니다.
2

코스(course) 기능

하나의 투어 안에서 스탬프존을 여러 묶음(코스)으로 나눠 보여주는 기능입니다. 여러 구역·테마·시장을 한 투어에서 운영할 때 유용합니다. 코스가 없으면 코스 칸을 전부 비워 두세요.

묶음 예시 — ‘시장별 맛집’ 투어
1코스 · 남대문시장
효자왕만두남대문숯불갈비대도식당
2코스 · 동대문시장
닭한마리골목엄마손칼국수
3코스 · 서문시장
서문칼제비납작만두
4코스 · 경동시장
경동빈대떡제기동순대국
코스 작성 규칙
  • 코스를 쓰는 경우, 같은 코스는 코스명을 완전히 똑같이 입력합니다(띄어쓰기·괄호·영문 포함).
  • 한 셀을 복사해 같은 코스의 다른 줄에 붙여넣으면 통일하기 쉽습니다.
  • 코스를 쓰지 않는 투어는 코스 칸을 전부 비워 둡니다.
⚠️ 한 글자만 달라도 다른 코스로 인식됩니다. 예) 1코스 ≠ 1 코스 (띄어쓰기) — 같은 묶음은 완전히 동일하게 입력하세요.
3

이미지 준비

이미지는 입력폼과 함께 ZIP의 images 폴더에 넣습니다. 스탬프투어는 한 장소에 여러 장(대표 + 추가)을 넣을 수 있는 점이 다릅니다.

이 사진이 들어가는 곳
🗺️
스탬프 지도지도 위 마커·미리보기
📋
스탬프 목록목록 카드 썸네일
🏛️
스탬프 상세대표 + 추가 사진(슬라이드)
권장 크기

스탬프 상세의 사진은 375 × 281 비율(가로가 살짝 긴 직사각형)을 권장합니다. 화면에서 잘려 보이지 않도록 이 비율에 맞춰 준비하면 가장 깔끔합니다.

파일명 규칙

이미지 파일은 번호 기준으로 저장하고, 엑셀의 번호 칸과 일치시킵니다.

파일명역할
1.png1번 장소의 대표 이미지(맨 처음 노출)
1_2.png · 1_3.png1번 장소의 추가 이미지(상세에서 순서대로 표시)
2.png2번 장소의 대표 이미지
(파일 없음)‘이미지 준비중’ 기본 이미지 자동 적용
지원 형식 · 그 외 안내
  • 지원 형식: png · jpg · jpeg · gif · webp
  • 한 장소에 사진이 여러 장이면 번호.png(대표) + 번호_2, _3 …(추가) 로 넣습니다.
  • 사진이 너무 무거우면(예: 1,000px 내외) 업로드·표시가 빠르고 균형이 좋습니다.
  • 홈 목록 썸네일·메인 배너 이미지는 입력폼과 별도로 관리자에서 설정합니다.
4

ZIP으로 묶어 업로드

엑셀 입력폼과 images 폴더를 하나의 ZIP으로 묶어 관리자에서 업로드합니다.

ZIP 구조
📦 stamptour.zip ├─ 스탬프투어_콘텐츠_입력폼.xlsx # 엑셀 입력폼 └─ 📁 images/ # 장소 이미지 폴더 ├─ 1.png # 1번 대표 ├─ 1_2.png # 1번 추가 ├─ 2.png └─ 3.png …
ZIP 만드는 방법
  • Windows — 엑셀 파일과 images 폴더를 함께 선택 → 우클릭 → ‘압축(ZIP 폴더로)’.
  • Mac — 두 항목 선택 → 우클릭 → ‘2개 항목 압축’.
  • 관리자 [스탬프 › 스탬프 일괄등록] → 대상 앱(스탬프투어) 선택 → ZIP 업로드 → 한 번에 등록.
💡 헤더명(번호·장소·…·코스)을 자동 인식하므로 고객 입력폼과 관리자 등록 항목이 그대로 일치합니다. 등록 후 앱(또는 미리보기)에서 지도 위 스탬프존 위치·반경과 코스 구분이 의도대로 보이는지 확인하세요.

자주 묻는 질문 (FAQ)

콘텐츠 입력·업로드 과정에서 자주 나오는 질문을 모았습니다.

이미지가 없어도 등록되나요?
네. 사진이 없는 스탬프존은 ‘이미지 준비중’ 기본 이미지가 자동으로 들어갑니다. 나중에 관리자에서 사진만 추가해도 됩니다.
위도·경도는 어디서 확인하나요?
지도(카카오맵·네이버지도·구글지도)에서 해당 지점을 길게 누르거나 우클릭하면 좌표가 나옵니다. 소수점 6자리 이상으로 입력할수록 위치가 정확합니다.
반경은 얼마로 설정하나요?
장소 크기에 맞춰 정합니다. 예) 공원·해변 100m / 일반 건물 50m / 작은 정자·묘역 20m. 너무 작으면 현장에서도 인증이 어렵고, 너무 크면 멀리서도 인증돼 부정 참여가 생길 수 있습니다.
코스는 꼭 입력해야 하나요?
아니요, 선택입니다. 스탬프존을 여러 묶음으로 나눌 때만 코스명을 적고, 나누지 않으면 코스 칸을 전부 비워 두세요. 같은 코스는 코스명을 완전히 똑같이 입력해야 같은 묶음으로 인식됩니다.
한 장소에 사진을 여러 장 넣을 수 있나요?
네. 번호.png가 대표 이미지, 번호_2.png·번호_3.png가 추가 이미지입니다. 상세 페이지에서 순서대로 표시됩니다.
유튜브·인스타 같은 외부 링크를 넣으려면?
장소 소개 칸에 전체 주소(https://…)를 넣으면 됩니다. 사용자가 누르면 해당 페이지로 이동합니다.
업로드했는데 일부만 등록되거나 실패했어요.
필수값(장소·장소 소개·위도·경도·반경)이 빠진 줄이 있으면 안전을 위해 전체가 취소되고, 어느 행이 문제인지 메시지로 알려줍니다. 해당 행을 고쳐 다시 업로드하면 됩니다.
엑셀의 항목 이름(헤더)을 바꿔도 되나요?
1행의 항목 이름은 그대로 두세요. 관리자가 헤더명을 자동 인식해 등록하므로, 이름을 바꾸면 인식되지 않을 수 있습니다. 열 순서가 바뀌는 것은 괜찮습니다.
파일 형식과 업로드 방법은?
엑셀(.xlsx)과 이미지(png·jpg·jpeg·gif·webp)를 한 폴더에 담아 ZIP으로 압축한 뒤, 관리자 [스탬프 › 스탬프 일괄등록]에서 대상 앱을 고르고 업로드합니다.
